4.      Explain why it does sense to break the modules into components? Try to give 4 different reasons with explanations. (2 points)

        Breaking modules into components enables developer to keep data and UI separate of each other.
        This further more enables to make the UI to be scalable depending on data size. Separation to components also provides better
        maintainability to the code base as now the small code pieces can be worked on by different developers without causing collisions
        as its unlikely same component needs to be worked by different devs at the same time.
